# 📦 Inventory Management System

> 🚀 A **Console-Based Inventory Management System** built using **Core Java**, designed to demonstrate **Object-Oriented Programming (OOP)**, **Java Collections Framework**, **Exception Handling**, and **In-Memory Data Management**.

---

## ✨ Features

✅ Add New Items (Perishable & Electronics)

✅ Update Stock (Add / Deduct)

✅ Search Items by ID

✅ Search Items by Name

✅ Display All Inventory Items

✅ View Low Stock / Reorder List

✅ Calculate Total Inventory Valuation

✅ Remove Existing Items

✅ Interactive Console-Based Menu

✅ In-Memory Data Storage using `HashMap`

✅ Custom Checked Exceptions

---
